# Contributing

1. Fork the repo and create a feature branch from `develop`.
2. Install deps: `make install`
3. Run tests before pushing: `make test`
4. Open a pull request against `develop`, not `main`.

## Code style

- Python: `black` (100 chars), `isort`
- Commit messages: `type(scope): subject` (conventional commits)
- No AI co-author lines in commits.
